地面座標 Line Trace

  • 地面進 Ground Channel
  • Line Trace 取 Hit.ImpactPoint
  • 回傳 Unreal 世界座標
  • 用來放物件 / 指定傳送點

傳送移動 NavMesh

  • NavMeshBoundsVolume 鋪可走區

  • 只能傳合法區
  • 落點高度正常、不卡牆

階段性程式碼

射線取得地面座標（C++）

Source/XR/XRGroundPointer.cpp
// 對照 Unity 版 Physics.Raycast(... groundMask)
bool UXRGroundPointer::TryGetGroundPoint(
    const FVector& RayOrigin, const FVector& RayDirection, FVector& OutGroundPoint) const
{
    FHitResult Hit;
    const FVector Start = RayOrigin;
    const FVector End = Start + RayDirection * TraceDistance;

    // ECC_GameTraceChannel1 = 自訂 Ground Trace Channel，別用 Visibility
    const bool bHitGround = GetWorld()->LineTraceSingleByChannel(
        Hit, Start, End, ECC_GameTraceChannel1);

    OutGroundPoint = bHitGround ? Hit.ImpactPoint : FVector::ZeroVector;
    return bHitGround;
}

藍圖版：Get Aim Transform → Line Trace By Channel(GroundTrace) → Break Hit Result → ImpactPoint。
