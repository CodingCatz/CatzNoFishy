核心觀念

兩層 SDK，分清楚

Meta XR Core SDK runtime

  • Quest runtime、OVRPlugin、平台能力
  • 負責「App 跑得起來」
  • 不提供抓取/射線/傳送

Meta XR Interaction SDK 互動

  • grab / ray / teleport / hand tracking
  • 負責「使用者能互動」
  • 只裝 Core 抓不到任何東西

套件選型

主推一套，備選一套

主推 · 只做 Quest Meta

  • Meta XR All-in-One SDK
  • + OpenXR + Interaction SDK
  • 現成 Rig / grab / ray / teleport
  • 手把與空手整合度最高

備選 · 要跨平台 Unity

  • XR Interaction Toolkit + XR Hands
  • 手把成熟，空手要自己串
  • hand ray / 手勢 select 多半得自接

建置設定

這幾項缺一個就 Build 不過

  • Build Profiles：Platform Android + Texture ASTC → Switch Platform
  • Scripting Backend = IL2CPP，Target Architectures = ARM64
  • Active Input Handling = Input System
  • Graphics API Vulkan 優先，別塞太多種避免環境不一致

四大互動，
其實是同一套骨架。

四大互動 ①

抓取 & 射線選取

抓取物件 Grabbable

  • Collider + Rigidbody + Grab Interactable
  • 手把：grip / trigger
  • 空手：pinch / palm
  • 遠距離才加 Ray / Distance Grab

射線選取 Ray Interactable

  • Collider + Ray Interactable / Pointable
  • hover 與 select 狀態
  • 事件回呼取得被選物
  • 手把射線、空手 hand ray 都可

四大互動 ②

地面座標 & 傳送

地面座標 Ground Layer

  • 地面進 Ground Layer
  • Physics.Raycast 配 groundMask
  • 拿到 hit.point 世界座標
  • 用來放物件 / 指定傳送點

傳送移動 Teleport

  • 可傳送區加 Teleport Interactable + Layer
  • 只能落在合法區
  • 落地高度正常、不卡牆
  • 手把搖桿、空手 L-gesture 都可

階段性程式碼

射線取得地面座標

Scripts/XR/GroundRaycaster.cs
public sealed class GroundRaycaster : MonoBehaviour
{
    [SerializeField] private Transform _rayOrigin;
    [SerializeField] private float _maxDistance = 100f;
    [SerializeField] private LayerMask _groundMask;   // 只命中地面

    public bool TryGetGroundPoint(out Vector3 groundPoint)
    {
        Transform origin = _rayOrigin != null ? _rayOrigin : transform;
        bool hitGround = Physics.Raycast(
            origin.position, origin.forward, out RaycastHit hit, _maxDistance, _groundMask);
        groundPoint = hitGround ? hit.point : Vector3.zero;
        return hitGround;
    }
}

SDK 給了射線元件，但「拿到地面點要幹嘛」是專案邏輯，得自己接。

雙路徑

手把 ↔ 空手，一個都不能少

手把 Controller

  • 抓取：Grip / Trigger
  • 射線：Controller Ray
  • 地座標：Controller Raycast
  • 傳送：Thumbstick / Button

空手 Hand

  • 抓取：Pinch / Palm Grab
  • 射線：Hand Ray
  • 地座標：Hand Raycast
  • 傳送：L-Gesture / Pinch

Layer 分層

射線該打到誰，靠 LayerMask 決定

  • 建 Layer：Ground / Interactable / Teleport / UI / Player
  • Object Ray Mask → 只命中可互動物
  • Ground Ray Mask → 只命中地面
  • Teleport Ray Mask → 只命中可傳送區，不誤判 UI 或抓取物
